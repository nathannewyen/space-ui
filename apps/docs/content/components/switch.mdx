# Switch

A binary on/off control with `role="switch"`.

## Anatomy

```tsx
import { Switch } from "@spacing-ui/core";

<Switch defaultChecked onCheckedChange={(c) => console.log(c)}>
  <span>Wi-Fi</span>
</Switch>;
```

## API

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` / `defaultChecked` | `boolean` | `false` | |
| `onCheckedChange` | `(checked: boolean) => void` | — | |
| `disabled` | `boolean` | `false` | |

## Accessibility

- `role="switch"`, `aria-checked="true"` or `"false"`
- Space and Enter toggle
- Disabled state via `aria-disabled` and `disabled`
